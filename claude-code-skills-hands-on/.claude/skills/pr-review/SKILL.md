# Skill: PR Review

## 概要

このスキルはプルリクエストのコードレビューを行う手順を定義します。

## チェック項目

### 1. 機能面

- 仕様通りに動くか
- エッジケースが考慮されているか
- エラーハンドリングがあるか

### 2. コード品質

- 関数が単一責務になっているか
- 変数名・関数名がわかりやすいか
- 重複コードがないか

### 3. セキュリティ

- 外部入力のバリデーションがあるか
- 機密情報がハードコードされていないか

## アウトプット形式

レビュー結果は以下の形式で出力してください。

**総評：** （一言）

**Good 👍**

- （良い点を箇条書き）

**要修正 🔧**

- （修正が必要な点を箇条書き）

**提案 💡**

- （あってもなくてもよい提案）
